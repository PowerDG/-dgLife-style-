**Affirmative sentences**

 

| If the same is true for you...?/td> |      | If the same is not true for you...?/td> |
| ----------------------------------- | ---- | --------------------------------------- |
| So + auxiliary + subject            |      | subject + (negative) auxiliary          |
| I'm very sociable.                  |      |                                         |
| So am I.                            |      | I'm not.                                |
